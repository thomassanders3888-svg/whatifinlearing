# NukeJobs.io

Nuclear careers for Navy nukes and operators.

## Features
- Salary calculator (Navy pay grade → civilian salary)
- Clearance bonus estimator
- SRO license premiums
- Featured job listings

## Target Audience
- Navy MMN, EMN, ETN ratings
- Commercial reactor operators
- SRO licensees
